## Импортированные инструкции (Mysterium)

### Старт работы с сервисом Mysterium

- Бот: https://t.me/MysteriumTechBot
- Базовый onboarding: подписка на канал, подтверждение в боте, выбор языка, принятие правил.
- В меню доступны: тестовый период 5 дней, покупка тарифа, пополнение баланса, партнерская ссылка, поддержка, инструкции.
- Покупка: "Купить подписку" -> выбрать тариф -> выбрать число устройств (по умолчанию 3, можно увеличить за доплату) -> "Продолжить" -> подтвердить заказ.
- Подключение: "Подключиться" -> мини-приложение -> выбрать ОС (iPhone, Android, macOS, Windows) -> пройти шаги.

### Как добавить подписку Mysterium на новое устройство

1. Открыть бота https://t.me/MysteriumTechBot.
2. Нажать "Подключиться".
3. В мини-приложении открыть экран с ссылкой и QR.
4. На новом устройстве: сканировать QR в V2RayTun или вставить ссылку через "+".
5. Выбрать страну и включить подключение.

### Как добавить подписку на macOS / Windows (Mysterium)

1. Открыть бота https://t.me/MysteriumTechBot.
2. Нажать "Подключиться".
3. Открыть окно с ссылкой и QR.
4. На ПК в V2RayTun нажать "+" и вставить ссылку из буфера.
5. Проверить подключение.

### Как добавить подписку на Android TV (Mysterium)

1. Установить V2RayTun на телевизор.
2. В ТВ-приложении: "Управление" -> "Импорт из QR-кода".
3. На телефоне в V2RayTun открыть QR для своей подписки.
4. Отсканировать QR с экрана ТВ.
5. Включить подключение на ТВ.

## Импортированные инструкции (Secretum)

### Как заказать новую подписку и добавить в приложение

1. В разделе "Подписки" нажать "Заказать Подписку".
2. Выбрать тариф.
3. Нажать "Пополнить баланс" (переход на YooMoney).
4. После оплаты вернуться в бот.
5. Далее: "Меню -> Подписки" -> выбрать активную подписку -> "Скачать приложение и подписку" -> "Добавить в iPhone / Android".

### Как добавить подписку в V2RayTun вручную

1. В боте: "Меню" -> "Подписки".
2. Выбрать подписку -> "Показать ссылку Подписки".
3. Скопировать ссылку.
4. В V2RayTun нажать "+" -> "Добавить из буфера обмена".
5. Выбрать подписку и включить подключение.

### Как добавить подписку на macOS / Windows (Secretum)

1. В боте: "Меню" -> "IT Подписки".
2. Выбрать подписку -> "Показать ссылку Подписки".
3. Скопировать ссылку.
4. В V2RayTun на ПК нажать "+" -> Import from clipboard.

### Как добавить подписку на Android TV (Secretum)

1. Установить V2RayTun на Android TV.
2. В приложении: "Управление" -> "Импорт из QR-кода".
3. На телефоне в V2RayTun показать QR для своей подписки.
4. Отсканировать QR на ТВ.
5. Включить подключение.

### Как удалить подписку

1. В боте: "Подписки" -> выбрать подписку -> "Удалить Подписку" -> "ДА УДАЛИТЬ".
2. В V2RayTun удалить профиль подписки вручную (удержание -> "Удалить").
