# Локальный AI-ассистент (Ollama)

## 1) Установите Ollama

- macOS: [https://ollama.com/download](https://ollama.com/download)

## 2) Скачайте модель

```bash
ollama pull llama3.1:8b
```

Можно выбрать другую модель и указать ее в `OLLAMA_MODEL`.

## 3) Заполните базу знаний

Редактируйте файл:

- `knowledge/service-instructions.md`

Там должны быть точные факты о тарифах, правилах и ответах поддержки.

## 4) Запустите проект

В одном терминале:

```bash
npm run llm:server
```

В другом терминале:

```bash
npm run dev
```

## 5) Проверьте работу

- Откройте сайт.
- Нажмите кнопку `AI-чат`.
- Отправьте вопрос.

Если сервер или Ollama не запущены, чат покажет подсказку об ошибке.
